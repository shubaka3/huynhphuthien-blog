Cấu trúc dữ liệu và thuật toán trong Java

Khám phá các cấu trúc dữ liệu phổ biến và thuật toán sắp xếp trong Java

Giới thiệu

Trong lập trình Java, việc hiểu rõ và sử dụng các cấu trúc dữ liệu như ArrayList, LinkedList, và HashMap là rất quan trọng. Những cấu trúc này giúp quản lý và thao tác dữ liệu hiệu quả hơn.

1. ArrayList

ArrayList là một danh sách động cho phép lưu trữ các phần tử có thể thay đổi kích thước. 

import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();
        list.add("Java");
        list.add("Python");
        list.add("C++");

        for (String lang : list) {
            System.out.println(lang);
        }
    }
}

2. LinkedList

LinkedList là danh sách liên kết kép, phù hợp với các thao tác thêm/xóa phần tử ở giữa danh sách. 

import java.util.LinkedList;

public class LinkedListExample {
    public static void main(String[] args) {
        LinkedList<Integer> numbers = new LinkedList<>();
        numbers.add(10);
        numbers.add(20);
        numbers.add(30);

        for (int num : numbers) {
            System.out.println(num);
        }
    }
}

3. HashMap

HashMap là một bảng băm cho phép lưu trữ cặp key-value, thường dùng để tra cứu nhanh. 

import java.util.HashMap;

public class HashMapExample {
    public static void main(String[] args) {
        HashMap<String, Integer> map = new HashMap<>();
        map.put("Alice", 25);
        map.put("Bob", 30);

        System.out.println("Age of Alice: " + map.get("Alice"));
    }
}

Thuật toán Bubble Sort

Thuật toán sắp xếp nổi bọt (Bubble Sort) là một thuật toán cơ bản giúp sắp xếp các phần tử trong mảng.

Ví dụ minh họa:

public class BubbleSort {
    public static void bubbleSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            for (int j = 0; j < n - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }

    public static void main(String[] args) {
        int[] arr = {64, 34, 25, 12, 22, 11, 90};
        bubbleSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}

Kết luận

Việc làm quen với các cấu trúc dữ liệu và thuật toán như Bubble Sort sẽ giúp bạn xây dựng các chương trình Java hiệu quả hơn. Hãy thực hành thường xuyên để nâng cao kỹ năng của mình.
