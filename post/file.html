Làm việc với file trong Java

Tìm hiểu cách đọc và ghi file trong Java.

1. Giới thiệu về làm việc với file trong Java

Java cung cấp các lớp trong thư viện java.io để xử lý việc đọc và ghi dữ liệu vào các file. Việc làm việc với file là một phần quan trọng trong lập trình vì nó giúp lưu trữ và quản lý dữ liệu lâu dài.

2. Cách ghi dữ liệu vào file

Để ghi dữ liệu vào file trong Java, bạn có thể sử dụng lớp FileWriter. Dưới đây là ví dụ về cách ghi chuỗi vào một file:

Mã nguồn ghi dữ liệu vào file:

import java.io.*;

public class FileExample {
    public static void main(String[] args) {
        try {
            FileWriter writer = new FileWriter("example.txt");
            writer.write("Hello, File!");
            writer.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

3. Cách đọc dữ liệu từ file

Để đọc dữ liệu từ file trong Java, bạn có thể sử dụng lớp FileReader. Dưới đây là ví dụ về cách đọc dữ liệu từ một file:

Mã nguồn đọc dữ liệu từ file:

import java.io.*;

public class FileExample {
    public static void main(String[] args) {
        try {
            FileReader reader = new FileReader("example.txt");
            int ch;
            while ((ch = reader.read()) != -1) {
                System.out.print((char) ch);
            }
            reader.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

4. Giải thích mã nguồn

Mã nguồn trên thực hiện hai thao tác cơ bản:

  • FileWriter: Sử dụng để ghi dữ liệu vào file. Nếu file không tồn tại, Java sẽ tạo mới file đó.
  • FileReader: Dùng để đọc dữ liệu từ file. Chúng ta đọc từng ký tự trong file cho đến khi hết nội dung.

5. Kết luận

Làm việc với file trong Java khá đơn giản nhờ vào các lớp tiện ích như FileWriter và FileReader. Bạn có thể áp dụng các lớp này để thực hiện các thao tác đọc và ghi file trong các ứng dụng Java của mình.
