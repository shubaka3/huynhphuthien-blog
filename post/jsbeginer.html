Học JavaScript cơ bản: Từ số không đến viết hàm đầu tiên

Khám phá các khái niệm cơ bản như biến, vòng lặp và cách viết hàm đầu tiên trong JavaScript!

1. Khái niệm cơ bản trong JavaScript

JavaScript là ngôn ngữ lập trình phổ biến được sử dụng để phát triển web. Dưới đây là một số khái niệm cơ bản bạn cần hiểu khi bắt đầu học JavaScript:

Biến (Variables)

Biến trong JavaScript là nơi bạn lưu trữ giá trị. Để khai báo một biến, bạn có thể sử dụng các từ khóa như var, let, hoặc const. Ví dụ:

let x = 10;

Biến x sẽ lưu trữ giá trị 10 trong ví dụ trên.

Vòng lặp (Loops)

Vòng lặp trong JavaScript giúp bạn lặp lại một đoạn mã nhiều lần. Ví dụ sử dụng vòng lặp for:

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Đoạn mã trên sẽ in ra các số từ 0 đến 4.

Hàm (Functions)

Hàm là một khối mã có thể tái sử dụng để thực hiện một công việc cụ thể. Bạn có thể khai báo hàm như sau:

function sum(a, b) {
    return a + b;
}

Hàm sum nhận hai tham số a và b, sau đó trả về tổng của chúng.

2. Ví dụ về hàm tính tổng

Để hiểu rõ hơn về cách viết hàm trong JavaScript, hãy thử ví dụ sau:

function sum(a, b) {
    return a + b;
}

console.log(sum(5, 10)); // Kết quả: 15

Trong ví dụ trên, hàm sum sẽ nhận hai số 5 và 10, sau đó trả về tổng là 15. Dòng lệnh console.log(sum(5, 10)); sẽ in ra kết quả này trên bảng điều khiển (console).

3. Kết luận

JavaScript là một ngôn ngữ lập trình mạnh mẽ và dễ học. Bằng cách hiểu rõ các khái niệm cơ bản như biến, vòng lặp và hàm, bạn sẽ có nền tảng vững chắc để phát triển các ứng dụng web. Hãy tiếp tục luyện tập và khám phá thêm nhiều tính năng mạnh mẽ của JavaScript!
