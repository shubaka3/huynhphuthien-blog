Học JavaScript cơ bản: Từ số không đến viết hàm đầu tiên

Khám phá các khái niệm cơ bản như biến, vòng lặp, điều kiện, hàm, và cách viết hàm đầu tiên trong JavaScript!

1. Khái niệm cơ bản trong JavaScript

JavaScript là ngôn ngữ lập trình phổ biến được sử dụng để phát triển web. Được sử dụng rộng rãi trong lập trình frontend (phía client) và backend (phía server), JavaScript giúp tạo ra các trang web động, tương tác với người dùng. Dưới đây là một số khái niệm cơ bản bạn cần hiểu khi bắt đầu học JavaScript:

Biến (Variables)

Biến trong JavaScript là nơi bạn lưu trữ giá trị. Bạn có thể khai báo một biến bằng cách sử dụng các từ khóa như var, let, hoặc const. Mỗi loại từ khóa có những điểm khác nhau:

  • var: Được sử dụng trong JavaScript từ các phiên bản cũ, có phạm vi toàn cục (global scope) hoặc phạm vi hàm (function scope).
  • let: Được giới thiệu trong ES6, có phạm vi khối (block scope), an toàn hơn khi làm việc với các vòng lặp.
  • const: Được sử dụng để khai báo hằng số, có giá trị không thể thay đổi sau khi được gán giá trị ban đầu.

Ví dụ:

let x = 10;

Biến x sẽ lưu trữ giá trị 10 trong ví dụ trên.

Vòng lặp (Loops)

Vòng lặp trong JavaScript giúp bạn lặp lại một đoạn mã nhiều lần. Đây là một kỹ thuật rất hữu ích khi bạn muốn thực hiện các thao tác lặp lại như duyệt qua danh sách, mảng, hoặc xử lý một số lượng lớn dữ liệu. Ví dụ sử dụng vòng lặp for:

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Đoạn mã trên sẽ in ra các số từ 0 đến 4.

Cũng có một số loại vòng lặp khác trong JavaScript như while và do...while giúp bạn linh hoạt hơn trong việc điều khiển vòng lặp:

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Hàm (Functions)

Hàm là một khối mã có thể tái sử dụng để thực hiện một công việc cụ thể. Bạn có thể khai báo hàm bằng từ khóa function trong JavaScript. Hàm nhận đầu vào là các tham số và có thể trả về một giá trị.

Ví dụ một hàm đơn giản tính tổng:

function sum(a, b) {
    return a + b;
}

Hàm sum nhận hai tham số a và b, sau đó trả về tổng của chúng. Bạn có thể gọi hàm này như sau:

console.log(sum(5, 10)); // Kết quả: 15

2. Các khái niệm nâng cao hơn trong JavaScript

Điều kiện (Conditionals)

Điều kiện trong JavaScript giúp bạn quyết định liệu một đoạn mã có thực thi hay không, dựa trên giá trị của các biểu thức. Bạn có thể sử dụng if, else, và else if để kiểm tra các điều kiện khác nhau.

let age = 18;
if (age >= 18) {
    console.log("Bạn đủ tuổi trưởng thành.");
} else {
    console.log("Bạn chưa đủ tuổi trưởng thành.");
}

Mảng (Arrays)

Mảng trong JavaScript là một danh sách có thể chứa nhiều giá trị. Bạn có thể tạo mảng và thao tác với các phần tử trong mảng bằng cách sử dụng chỉ số (index). Mảng có thể chứa bất kỳ loại dữ liệu nào, bao gồm cả chuỗi, số, hoặc các đối tượng phức tạp.

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[1]); // Kết quả: Banana

Trong ví dụ trên, mảng fruits chứa ba loại trái cây, và bạn có thể truy cập phần tử thứ hai bằng cách sử dụng chỉ số 1.

Đối tượng (Objects)

Đối tượng trong JavaScript cho phép bạn nhóm các thuộc tính và phương thức lại với nhau dưới một tên chung. Một đối tượng có thể chứa các thuộc tính (key) và giá trị (value), và các phương thức là các hàm thuộc về đối tượng đó.

let person = {
    name: "John",
    age: 30,
    greet: function() {
        console.log("Hello " + this.name);
    }
};
person.greet(); // Kết quả: Hello John

Đoạn mã trên khai báo một đối tượng person với các thuộc tính name, age, và phương thức greet.

3. Ví dụ về hàm tính tổng

Để hiểu rõ hơn về cách viết hàm trong JavaScript, hãy thử ví dụ sau:

function sum(a, b) {
    return a + b;
}

console.log(sum(5, 10)); // Kết quả: 15

Trong ví dụ trên, hàm sum sẽ nhận hai số 5 và 10, sau đó trả về tổng là 15. Dòng lệnh console.log(sum(5, 10)); sẽ in ra kết quả này trên bảng điều khiển (console).

4. Kết luận

JavaScript là một ngôn ngữ lập trình mạnh mẽ và dễ học. Bằng cách hiểu rõ các khái niệm cơ bản như biến, vòng lặp, điều kiện, mảng, đối tượng và hàm, bạn sẽ có nền tảng vững chắc để phát triển các ứng dụng web động. Hãy tiếp tục luyện tập và khám phá thêm nhiều tính năng mạnh mẽ của JavaScript!
