Spring Boot: Làm ứng dụng web cơ bản

Hướng dẫn tạo REST API đơn giản với Spring Boot

Giới thiệu Spring Boot

Spring Boot là một framework mạnh mẽ giúp xây dựng các ứng dụng web và REST API nhanh chóng với cấu hình tối thiểu. Với Spring Boot, bạn có thể tập trung vào logic của ứng dụng mà không phải lo lắng quá nhiều về cấu hình chi tiết.

Cách tạo REST API đơn giản

Một REST API cơ bản trong Spring Boot chỉ cần vài bước đơn giản:

  1. Khởi tạo một project Spring Boot.
  2. Thêm các dependencies cần thiết (Spring Web).
  3. Viết controller để xử lý các yêu cầu HTTP.

Ví dụ Code:

@RestController
public class HelloController {
    @GetMapping("/hello")
    public String sayHello() {
        return "Hello, Spring Boot!";
    }
}

Hướng dẫn thực hiện

Thực hiện các bước sau để tạo ứng dụng REST API đơn giản:

1. Khởi tạo dự án

Truy cập trang Spring Initializr và tạo một dự án với các thông tin sau:

  • Project: Maven
  • Language: Java
  • Spring Boot Version: 3.x.x
  • Dependencies: Spring Web

2. Cấu trúc dự án

Sau khi tải xuống, giải nén và mở dự án trong IDE yêu thích (IntelliJ, Eclipse). Thêm class HelloController trong package chính để định nghĩa REST API.

3. Chạy ứng dụng

Chạy file Application.java. Sau khi khởi động thành công, mở trình duyệt và truy cập URL http://localhost:8080/hello. Bạn sẽ thấy kết quả: 

Hello, Spring Boot!

Kết luận

Spring Boot giúp việc phát triển các ứng dụng web và REST API trở nên đơn giản hơn bao giờ hết. Với các tính năng mạnh mẽ và linh hoạt, đây là một công cụ không thể thiếu cho các lập trình viên Java hiện đại.
