OOP trong Java: Học qua ví dụ thực tế

Hiểu về Lập trình Hướng đối tượng thông qua các ví dụ dễ hiểu

Giới thiệu về OOP

Lập trình Hướng đối tượng (Object-Oriented Programming - OOP) là một phương pháp lập trình dựa trên khái niệm "đối tượng", trong đó mỗi đối tượng có thuộc tính (dữ liệu) và phương thức (hành vi).

Các khái niệm chính trong OOP

  • Class: Là bản thiết kế hoặc khuôn mẫu cho các đối tượng.
  • Object: Là một thể hiện của class.
  • Inheritance: Là tính kế thừa, cho phép một class sử dụng lại các thuộc tính và phương thức của class khác.
  • Polymorphism: Là tính đa hình, cho phép cùng một phương thức có thể có các cách thực hiện khác nhau.

Ví dụ: Quản lý sinh viên sử dụng OOP

Code:

class Student {
    String name;
    int age;

    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public void displayInfo() {
        System.out.println("Name: " + name + ", Age: " + age);
    }
}

public class Main {
    public static void main(String[] args) {
        Student student = new Student("John", 20);
        student.displayInfo();
    }
}

Phân tích code

Trong ví dụ trên:

  • Class Student: Định nghĩa thuộc tính name và age, cùng với phương thức displayInfo.
  • Constructor: Khởi tạo đối tượng Student với tên và tuổi.
  • Object: Đối tượng student được tạo từ class Student.

Ý nghĩa của OOP

OOP giúp lập trình viên quản lý và tổ chức code tốt hơn, đặc biệt trong các dự án lớn. Nó cũng giúp tái sử dụng code và giảm thiểu lỗi bằng cách áp dụng các nguyên tắc rõ ràng.

Kết luận

Lập trình Hướng đối tượng là một phần không thể thiếu khi học Java. Hãy thực hành các ví dụ thực tế để nắm vững các khái niệm cơ bản và nâng cao kỹ năng lập trình của bạn.
