DOM Manipulation: Làm quen với lập trình giao diện

Khám phá cách sử dụng JavaScript để thay đổi nội dung và tương tác với trang web của bạn!

1. Giới thiệu về DOM Manipulation

DOM (Document Object Model) là một API (Application Programming Interface) cho phép JavaScript tương tác và thay đổi nội dung của trang web. Với DOM, bạn có thể thay đổi văn bản, thay đổi cấu trúc HTML, hoặc thay đổi các thuộc tính của phần tử trang web.

Thay đổi nội dung trang web bằng JavaScript

JavaScript cung cấp các phương thức để thay đổi nội dung của các phần tử HTML trên trang web. Bạn có thể sử dụng document.getElementById() để lấy phần tử HTML và sau đó thay đổi thuộc tính của nó, như thay đổi văn bản hoặc giá trị của thuộc tính.

2. Ví dụ về thay đổi nội dung HTML

Ví dụ dưới đây cho thấy cách thay đổi nội dung của một phần tử HTML khi người dùng nhấn vào một nút:

Mã HTML:

<button id="myButton">Click me</button>
<p id="output"></p>

Mã JavaScript:

document.getElementById("myButton").addEventListener("click", function() {
    document.getElementById("output").innerText = "Button clicked!";
});

Trong ví dụ trên, khi người dùng nhấn vào nút có ID myButton, nội dung của phần tử output sẽ thay đổi thành "Button clicked!".

3. Kết luận

DOM Manipulation là một kỹ năng quan trọng khi làm việc với JavaScript để thay đổi nội dung và giao diện của trang web. Bạn có thể sử dụng các phương thức như getElementById, getElementsByClassName, querySelector, v.v., để tương tác với các phần tử trong DOM. Hãy tiếp tục khám phá và thử nghiệm với các tính năng của JavaScript để làm phong phú trang web của bạn!
